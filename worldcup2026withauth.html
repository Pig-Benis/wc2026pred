One-time admin setup

Set up your predictor

You need a free Firebase project (by Google) to store predictions. It's free forever, takes about 3 minutes, and your friends won't need to do anything — they just open the link.

1
Go to console.firebase.google.com and click Create a project. Give it any name. Disable Google Analytics when asked.
2
Once created, click Build → Realtime Database in the left sidebar, then Create Database. Choose any location, and select Start in test mode.
3
Copy the database URL — it looks like:
https://your-project-default-rtdb.firebaseio.com
4
Go to Project Settings (gear icon) → scroll to Your apps → click </> Web → register app (any name) → copy the apiKey and databaseURL from the config snippet.

Paste your Firebase config below. These get baked into the file — your friends won't see or need them.

All set!

One-time admin setup

Set up your predictor

You need a free Firebase project to store predictions. Takes ~3 minutes. Your friends just open the link and pick a name — nothing else needed.

1
Go to console.firebase.google.com → Create a project. Any name. Disable Analytics.
2
Click Build → Realtime Database → Create Database. Pick any location, choose Start in test mode.
3
Go to Project Settings (gear ⚙ next to project name) → scroll to Your apps → click </> → register web app → copy apiKey, databaseURL, projectId.

Paste your Firebase config. These get baked into the downloaded file — your friends won't see them.

All set!
